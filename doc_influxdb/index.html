InfluxDB v1.11 文档学习笔记

主要特点

  • 专为时间序列数据编写的高性能数据存储。高速读取和数据压缩。
  • 为您的集群提供高可用性并消除单点故障。
  • 完全用 Go 编写。编译为单个二进制文件，无任何外部依赖。
  • 简单、高性能的写入和查询 HTTP API。
  • 插件支持其他数据提取协议，例如 Graphite、collectd 和 OpenTSDB。
  • 富有表现力的类似 SQL 的查询语言，用于轻松查询聚合数据。
  • 连续查询会自动计算聚合数据，使频繁查询更加高效。
  • 标签让您可以索引系列以实现快速高效的查询。
  • 保留策略可以有效地自动使陈旧数据过期。

时间序列预测模型

3种技术对比

维度 Prophet NeuralProphet Chronos-2
开发方 Meta (Facebook) Meta 研究团队 Amazon Science
模型类型 统计模型（可加/可乘分解） 深度学习（AR-Net + Prophet 分解） 基础模型（预训练 Transformer）
底层框架 Stan (CmdStanPy) PyTorch PyTorch + HuggingFace Transformers
训练方式 每次从零拟合 每次从零训练 预训练 + 零样本推理；可选微调
零样本能力 无 无 支持，开箱即用
微调 — — 支持全量微调 & LoRA 微调
GPU 需求 不需要 可选（PyTorch 自动适配） 推荐（CPU 可用但较慢）
协变量支持 `add_regressor` 添加未来已知协变量 未来协变量 + 滞后协变量 未来已知协变量（`covariate_tensors`）
概率预测 内置置信区间（`yhat_lower/upper`） 内置分位数 原生多采样路径，输出分位数区间
可解释性 高：趋势 / 季节性 / 节假日可分解 中：支持组件分解图 低：黑盒神经网络
调参复杂度 低：少量超参（changepoint / seasonality） 中：学习率、epochs、AR 阶数等 低：零样本无需调参；微调需设定 epochs / LoRA rank
API 易用性 高：fit → predict 两步完成 高：fit → predict，API 风格类似 Prophet 高：from_pretrained → predict 两步完成
训练速度 快（秒级） 中等（秒 ~ 分钟级，取决于数据量与 epochs） 零样本秒级推理；微调分钟 ~ 小时级
适用场景 趋势 + 季节性明显、数据量中等 需要自回归建模或非线性拟合 数据稀缺零样本、多场景快速验证、微调追求极致精度

# 使用python虚拟环境防止依赖冲突

source .venv/bin/activate # 进入虚拟环境
deactivate                # 退出虚拟环境

# 安装依赖，会存储在 .venv 虚拟环境目录中
pip install chronos-forecasting
pip install pyarrow
pip install fastparquet
pip install matplotli
pip install ipywidgets

# 固定依赖版本
pip freeze > requirements.txt

# 导出成markdown
pip install jupyter
jupyter nbconvert --to markdown chronos2-example.ipynb

问题解决

chronos 示例中第一次执行 from_pretrained 函数卡住

因为在下载模型，需要科学上网或者使用https://hf-mirror.com等国内镜像站下载，模型默认存储到用户缓存目录，如: ~/.cache/huggingface/,下一次执行会先从缓存读取无需重新下载。

chronos 示例中 pyarrow 模块内出现重复定义报错

ArrowKeyError: A type extension with name pandas.period already defined

这是一个已知的兼容性问题，可以指定使用 fastparquet 引擎规避该问题

energy_context_df = pd.read_parquet(
    "https://autogluon.s3.amazonaws.com/datasets/timeseries/electricity_price/train.parquet",
    engine="fastparquet"
)

图表中文字体乱码

# 安装字体
sudo apt install ttf-wqy-zenhei ttf-wqy-microhei

# 检查字体是否安装
fc-list | grep -i "wenquanyi"

# 删除字体缓存
rm ~/.cache/matplotlib/fontlist-*.json

# 或者显式注册字体文件，避免 matplotlib 缓存未收录时报 findfont 警告
# fm.fontManager.addfont('/usr/share/fonts/truetype/wqy/wqy-zenhei.ttc')
# fm.fontManager.addfont('/usr/share/fonts/truetype/wqy/wqy-microhei.ttc')

# 加载中文字体
plt.rcParams['font.sans-serif'] = ['WenQuanYi Zen Hei', 'WenQuanYi Micro Hei']
plt.rcParams['axes.unicode_minus'] = False

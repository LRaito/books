sync.Once 代码解读

go 版本: 1.22.2

设计思想

使用 Mutex 加锁，用 double check 和原子量做标记

关键结构

type Once struct {
    done atomic.Uint32 // 标记位，调用一次后标记成1
    m    Mutex         // 互斥量
}

代码解读

func (o *Once) Do(f func()) {
    if o.done.Load() == 0 {
        o.doSlow(f)
    }
}

func (o *Once) doSlow(f func()) {
    o.m.Lock()
    defer o.m.Unlock()

    // double check 后执行函数，再更新标记
    if o.done.Load() == 0 {
        defer o.done.Store(1)
        f()
    }
}

插件安装

插件安装

1.安装插件

使用 npm 安装插件，命令参数为:gitbook-plugin-插件名称 以目录折叠插件示例，如果是容器化部署可以直接在容器内执行：

npm install gitbook-plugin-toggle-chapters

2.编辑配置文件

项目根目录下需要有文件 bool.json，示例如下：

{
    "plugins": [
        "toggle-chapters"
    ]
}

3.插件生效

GitBook 支持热更新，修改后会自动生效
